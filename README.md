# More Tags
*A CSS & JS Library*

<br>

### `<mono>`
Makes all child text monospaced.  
**HTML Example:** `<mono>Code</mono>`

<br>

### `<log id="[logName]">`
User viewable log accessed via the `log()` object.  
**HTML Example:** `<log id="test"></log>`  
**JS Example:** `var testLog = new Log('test'); testLog.print('Hello, World!');`
